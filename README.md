# Amazon Product Analysis

## Project Overview

This project analyzes Amazon product data to identify product trends, discounts, ratings, category distribution, customer preference

## Dataset

* Total Products: 1465
* Features: Product Name, Category, Rating, Discount Percentage, Price, Review Count

## Technologies Used

* Python
* Pandas
* Matplotlib
* VS Code

## Analysis Performed

1. Data Cleaning
2. Missing Value Handling
3. Category Analysis
4. Product Rating Analysis
5. Discount Analysis
6. Review Count Analysis

## Visualizations

* Top Product Categories
* Most Reviewed Products
* Highest Discounted Products

## Key Insights

* Certain categories dominate Amazon listings.
* Some products offer extremely high discounts.
* Highly reviewed products generally maintain strong ratings.
* Customer ratings vary significantly across categories.

## Skills Demonstrated

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Data Visualization
* Business Insight Generation
* Python Programming
